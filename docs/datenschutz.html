2. Datenerfassung in der App

Die Rosenkranz-App ist eine reine Offline-App. Es werden keine personenbezogenen Daten erhoben, verarbeitet oder genutzt. Insbesondere:

  • Keine Analyse-Tools (Google Analytics, Firebase oder ähnliche)
  • Keine Tracker oder Cookies
  • Keine Werbung (keine Ad-Netzwerke eingebunden)
  • Keine Cloud-Synchronisation (alle Daten bleiben lokal)
  • Keine Registrierung oder Nutzerkonten erforderlich
  • Keine Weitergabe von Daten an Dritte

3. Lokale Speicherung

Die App speichert folgende Informationen ausschließlich lokal auf deinem Gerät:

  • Einstellungen: Sprachauswahl, Anzeigegeschwindigkeit und aktiver Rosenkranz-Tag
  • News-Gelesen-Status: Welche Neuigkeiten du bereits gesehen hast

Diese Daten werden über die SharedPreferences (Android/iOS) bzw. localStorage (Web) gespeichert. Sie können jederzeit über die App-Einstellungen oder durch Löschen der App-Daten entfernt werden.

4. App-Updates

Die App prüft in regelmäßigen Abständen (alle 24 Stunden) auf dem GitHub-Server, ob eine neue Version verfügbar ist. Dabei wird eine einfache HTTP-Anfrage an den Server gestellt. Hierbei werden keine personenbezogenen Daten übermittelt. Der Server empfängt lediglich die IP-Adresse, was bei jeder Internetverbindung technisch bedingt ist.

5. Website (Webseite & Handbuch)

Diese Website wird statisch auf GitHub Pages gehostet. GitHub kann als Hosting-Anbieter technische Zugriffsdaten (insbesondere die IP-Adresse) erfassen. Für die Datenverarbeitung durch GitHub gelten die Datenschutzbestimmungen von GitHub.

Es werden auf dieser Website keine Cookies gesetzt, keine Tracking-Skripte eingebunden und keine Analysedienste (wie Google Analytics) verwendet.

6. Verlinkungen zu Diensten Dritter

Die Website kann Links zu externen Diensten (z. B. Google Play Store, Apple App Store) enthalten. Sofern die App in Zukunft in diesen Stores verfügbar ist, führen die Links dorthin. Wenn du diesen Links folgst, werden die jeweiligen Betreiber deine IP-Adresse und ggf. weitere technische Daten empfangen. Für die Datenverarbeitung durch diese Dienste gelten die jeweiligen Datenschutzbestimmungen der Anbieter.

Derzeit ist die App weder im Google Play Store noch im Apple App Store gelistet. Die entsprechenden Buttons auf der Website sind als Vorankündigung gekennzeichnet.
